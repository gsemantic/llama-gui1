# Система примагничивания окон по сетке (Grid Snapping)

## Обзор

Система примагничивания окон по сетке обеспечивает выравнивание положения и размеров окон по невидимой сетке, подобно тому как это реализовано в Adobe Photoshop и других графических редакторах.

## Возможности

### Основные функции
- **Примагничивание позиции** - выравнивание положения окон по сетке при перемещении
- **Примагничивание размеров** - выравнивание размеров окон при изменении
- **Два режима работы**:
  - Основная сетка (по умолчанию 16 пикселей)
  - Точная подстройка (fine-tuning) с делителем (по умолчанию 16/4 = 4 пикселя)

### Настройки
- **Размер сетки** (4-64 пикселя)
- **Включение/выключение примагничивания**
- **Порог примагничивания** (1-32 пикселя)
- **Визуализация сетки** (отладочный режим)

## Использование

### Через меню
1. Откройте меню **Вид** → **Примагничивание по сетке...**
2. В диалоге настроек:
   - Включите примагничивание
   - Настройте размер сетки
   - При необходимости включите точную подстройку
   - Нажмите "Примагнитить все окна" для немедленного применения

### Через команды
```bash
# Включить/выключить примагничивание
toggle_grid_snapping

# Примагнитить все окна к сетке
snap_windows_to_grid

# Показать/скрыть визуальную сетку
toggle_grid_overlay

# Открыть диалог настроек
show_grid_snapping_dialog
```

## Производительность

**Минимальное влияние на производительность** (< 0.1% CPU):

- Вычисления выполняются только при перемещении/изменении размера окна
- Используются простые математические операции: `round(value / grid_size) * grid_size`
- Кэширование результатов в рамках одного кадра
- **Не влияет на скорость генерации токенов**

## Архитектура

### Компоненты

1. **GridSnappingSystem** (`include/ui/grid_snapping_system.h`)
   - Основная логика примагничивания
   - Настройки системы
   - Визуализация сетки

2. **GridSnappingDialog** (`include/ui/grid_snapping_dialog.h`)
   - Диалог настроек
   - Интерфейс пользователя

3. **WindowManager** (расширен)
   - Интеграция с системой примагничивания
   - Методы `snapWindowToGrid()`, `snapAllWindowsToGrid()`

### Интеграция

Система интегрирована в:
- `MainWindow::render_main_layout()` - отрисовка сетки
- `MainWindow::render_main_window_rendering_core.cpp` - применение примагничивания
- `CommandManager` - команды управления
- `AdvancedMenuSystem` - пункты меню

## Примеры настроек

### Для точного выравнивания (как в Photoshop)
```
Размер сетки: 16 px
Точная подстройка: Включена
Делитель: 4
→ Мелкая сетка: 4 px
```

### Для быстрого расположения окон
```
Размер сетки: 32 px
Точная подстройка: Выключена
→ Основная сетка: 32 px
```

### Для свободной расстановки с лёгким примагничиванием
```
Размер сетки: 20 px
Порог примагничивания: 16 px
Точная подстройка: Включена
Делитель: 2
→ Мелкая сетка: 10 px
```
